## 🚀 Featured Projects

### 🔐 Room Security System Using Arduino and C++
- Developed a smart room security system using Arduino and C++.
- Integrated PIR motion sensor, DHT11 sensor, buzzer, LED, and servo motor.
- Designed, simulated, and tested the system using Tinkercad and hardware implementation.

### 🚨 Motion Detection Alarm System
- Designed and implemented an Arduino-based motion detection alarm system.
- Integrated PIR sensors and electronic components to detect motion and trigger alerts.

### ⚡ DC Power Supply
- Designed and implemented a regulated DC power supply circuit.
- Applied rectification, filtering, and voltage regulation techniques.
- Tested and verified stable DC output through hardware implementation.

### 🔢 3-Bit Synchronous Binary Counter
- Designed and implemented a synchronous counter using JK flip-flops.
- Applied truth tables, excitation tables, and K-map simplification.
- Verified outputs through simulation and hardware testing.

### ➕ 2-Bit Arithmetic and Logic Unit (ALU)
- Designed a 2-bit ALU capable of performing arithmetic and logic operations.
- Implemented operations such as addition, subtraction, AND, OR, XOR, and NOT.
- Verified functionality through simulation and breadboard implementation.

### 🎵 Voice Signal Processing using MATLAB
- Processed and analyzed voice signals in time and frequency domains.
- Applied FFT, echo generation, signal reversal, and tone addition.
- Visualized and saved processed audio signals in WAV format.
